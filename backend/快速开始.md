# 🚀 API接口管理工具 - 快速开始

## 📋 问题解决

您的系统有**70+个API接口**，现在可以通过以下工具轻松管理：

### ✅ 解决的问题
- ✅ **快速了解所有接口状态** - 一键批量检测
- ✅ **方便手动调用接口** - 交互式测试
- ✅ **查看API文档** - 即使在生产环境
- ✅ **监控系统健康** - 自动化检查

## 🛠️ 工具使用

### 1. 轻量级API测试器 (推荐)

**特点**: 无需安装任何依赖，开箱即用

```bash
cd backend

# 快速健康检查 (推荐先用这个)
python3 api_tester_simple.py --mode health

# 批量测试所有接口
python3 api_tester_simple.py --mode batch

# 交互式测试 (最实用)
python3 api_tester_simple.py --mode interactive

# 生成健康报告
python3 api_tester_simple.py --mode report

# 测试远程服务器
python3 api_tester_simple.py --url https://your-api.com --mode health
```

### 2. 现代化API管理页面 (推荐)

```bash
# 启动后端服务器
python3 run.py
```

然后访问:
- 🎨 **API管理控制台**: http://localhost:8000/api-manager
- 📚 **Swagger文档**: http://localhost:8000/docs (如果启用)
- 📖 **ReDoc文档**: http://localhost:8000/redoc (如果启用)

### 3. 临时启用Swagger文档

```bash
# 临时启用Swagger文档 (即使在生产环境)
python3 enable_docs.py
```

## 🔧 高级用法

### 定时监控
```bash
# 添加到crontab，每小时检查一次
0 * * * * cd /path/to/backend && python3 api_tester_simple.py --mode health >> logs/health.log 2>&1
```

### 部署流程集成
```bash
# 在部署脚本中添加
python3 api_tester_simple.py --mode batch
if [ $? -eq 0 ]; then
    echo "✅ API测试通过，部署成功"
else
    echo "❌ API测试失败，回滚部署"
    exit 1
fi
```

### 开发环境测试
```bash
# 启动开发服务器后
python3 api_tester_simple.py --url http://localhost:8000 --mode health
```
