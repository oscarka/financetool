### 1. 🎨 现代化Web界面 (最推荐)

**特点**: 美观、直观、功能强大

```bash

cd backend && python3 run.py

# 访问地址
http://localhost:8000/api-manager
```

**功能**:
- ✅ 美观的渐变背景和毛玻璃效果
- ✅ 分类展示所有32个接口
- ✅ 一键点击测试任何接口
- ✅ 实时显示测试结果
- ✅ 移动端完美适配
- ✅ 完全中文化界面

### 2. 🛠️ 命令行工具 (功能全面)

**特点**: 功能强大、可自动化

```bash

python3 api_tester_simple.py --mode health

# 批量测试所有接口
python3 api_tester_simple.py --mode batch

# 交互式测试
python3 api_tester_simple.py --mode interactive

# 生成健康报告

```

**功能**:
- ✅ 批量测试所有接口
- ✅ 交互式调用
- ✅ 生成详细报告
- ✅ 支持远程服务器
- ✅ 无需额外依赖
